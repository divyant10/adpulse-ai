# 🚀 AdPulse AI — Advertising Diagnostics Platform

AdPulse AI is an **AI-powered advertising analytics platform** that analyzes campaign performance, detects anomalies, calculates campaign health scores, and generates **AI-driven optimization strategies**.

The platform simulates the type of **diagnostic intelligence used in modern advertising systems like Google Ads and Meta Ads Manager.**

# 🧠 Problem

Digital advertising platforms generate massive amounts of campaign data:

* impressions
* clicks
* CTR
* CPC
* conversions

However, marketers struggle to answer key questions:

• Which campaigns are wasting budget?
• Why is CTR dropping?
• Which campaigns should be scaled?
• Where are anomalies occurring?

Most dashboards only show **raw analytics**, not **actionable intelligence**.

AdPulse AI solves this by combining **analytics + AI diagnostics**.

---

# ⚡ Features

## 📊 Campaign Analytics Dashboard

* Real-time campaign performance overview
* CTR, CPC, and conversion analytics
* Interactive visualizations

## 🧮 Campaign Health Score

A Google Ads–style optimization score that evaluates campaign performance on a **0–100 scale** using:

* CTR performance
* CPC efficiency
* Conversion rate

## 📉 Campaign Optimization Engine

Automatically identifies:

* 🚫 Campaigns to **pause**
* 🚀 Campaigns to **scale**
* 💸 Campaigns **wasting budget**

## 🚨 Anomaly Detection

Detects abnormal campaign behavior such as:

* CTR drops
* CPC spikes
* conversion crashes

## 🤖 AI Marketing Insights

Uses **Gemini AI** to generate campaign optimization strategies.

Example insights:

* Improve ad creatives to increase CTR
* Reduce high CPC keywords
* Focus budget on high-converting campaigns
* Fix tracking errors causing conversion inaccuracies

## 💬 AI Assistant

Interactive AI chat interface for marketing diagnostics.

Example queries:

* *Which campaigns should I scale?*
* *Why is my CTR low?*
* *What optimization strategies should I apply?*

## 📁 Dataset Upload

Users can upload custom datasets to analyze their own campaign data.

# 📦 Installation

Clone the repository:

```bash
git clone https://github.com/divyant10/adpulse-ai.git
cd adpulse-ai
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run dashboard/app.py
```

---

# 🔐 Environment Variables

Add your Gemini API key:

```
GEMINI_API_KEY=your_api_key
```
